Account window menus

The menu is the start of every actions on transactions.

Edit

  • Filter... - open the filter dialog

Transaction

  • Add... - open the transaction dialog in Add mode
  • Inherit... - open the transaction dialog prefilling it with the selected transaction
  • Edit... - open the transaction dialog in Edit mode
  • (De)Validate... - toggle the transaction validated state
  • Remove... - remove the selected transaction(s)
  • Make an archive... - make archive with the selected transaction(s)

Tools

  • Import csv... - import transaction from a csv file, see CSV file format
  • Export csv... - export transaction to a csv file, see CSV file format
