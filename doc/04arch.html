Archives dialog

The archives dialog is used to add, edit and manage the archives of the current wallet.

Dialog call

This dialog can be called in the following way:
  • from the mainwindow-menu: Manage / Archives...
  • from the mainwindow-toolbar.

Using archives dialog

Archive list

display the actual list of archives in the wallet. It is always sorted in alphabetic order.

Buttons

Remove remove the active archive.
Add add a new empty archive.

General infos / Optional infos

Please refer to transaction dialog for these fields.

Automated insertion

Activate set this archive to be automated
Limit to limit the insertion to a finished count
times number of insertion to limit to
Every set the insertion interval
Units set the unit of the insertion interval: day, week, month, year
Next on specify the date of the next insertion, when you first edit you should set it manually, then it is updated automatically. You can of course adjust it later if needed.
