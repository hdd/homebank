Accounts dialog

The account dialog is used to add, edit and manage the accounts of the current wallet.

Dialog call

This dialog can be called in the following way:
  • from the mainwindow-menu: File / Properties
  • from the mainwindow-toolbar.

Using wallet dialog

Account list

display the actual list of accounts in the wallet. The account at the top of the list will be the one to be displayed when opening your wallet. You can easily change the accounts order using Drag&Drop process.

Buttons

Remove remove the active account. This is only possible if the account has no transactions.
Add add a new empty account.

Informations

Heading specify a name that identify the account.
Number specify the bank account number itself.
Bank name specify the bank name where the account is registered.
Include in the budget specify to include this account into the budget evaluation.
This account was closed specify that the account is closed.

Balances

Initial specify the initial balance amount, i.e the balance before the first transaction.
Overdrawn at specify the overdrawn threshold, if you have an authorized overdrawn, this will be used in the Overdrawn report window.

Cheque number

Notebook 1 specify the next cheque number of pad1 to be used (this is auto updated later when you add some transactions).
Notebook 2 some of the above but for a second cheque pad (for a joined account)
