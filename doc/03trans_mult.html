Multiple transaction edition

Homebank enable to change the value of a single column for multiple transaction at the same time.
This can be done in the account window.

How to use ?

  • You must first select several transaction, what is usually done by holding the <shift> key while clicking on some transaction.
  • Then, while holding the <shift> key, just double click on the column to change the value.
  • (the transaction on which you click will serve to initialize the value of the pop-up).
  • A pop-up will rise, (showing the value of the transaction you clicked)
  • Then you can change that value and apply it to the selected transactions.

Column available

All column support that great feature (except the 'State' column):

  • Date
  • Info
  • Payee
  • Description
  • Amount
  • Category
  • Tags

Step by step example

Let's say you want to change the 'Payee' of several transactions:

  • select multiple transaction you want to change payee (usually with shift key)
  • while holding shift key, double click on the payee column
  • a dialog opens, pre filled with the value of the transaction you double clicked on
  • change the value
  • validate
  • it's magic !
