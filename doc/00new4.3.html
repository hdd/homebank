New features in v4.3

  • Welcome dialog
  • Wallet assistant
  • Preset categories

:: Welcome dialog

When you first run Homebank, you will be asked from a dialog for what to do:

  • Create a new wallet
  • Open an existing wallet
  • Open the example file

Later, you still can access to this dialog from the Help menu.

:: Wallet assistant

Now, when you want to create a new wallet, an assistant will help you to configure some minimal parameters.

  • The owner name
  • Import some preset categories (see below) for your language (if the file exist)
  • Create your 1st account

:: Preset categories

This is something I'd like to implement a long time ago, but this time it is. When you create a new wallet, with the new assistant, Homebank will fill some default categories for you.

The preset files exists for the following languages, and are based categories from microsoft money:

  • de - Deutsch
  • en - English (native)
  • es - Spanish
  • fr - French (native)
  • it - Italian
  • pt - Portuguese
  • ro - Romanian
  • ru - Russian

(non native were translated from google translate and may be inacurrate)

These files are simple HomeBank native CSV category files (same as those from export button) and are located in:

  • Linux: /usr/share/homebank/datas
  • Windows: <installdir>/share/homebank/datas (usually C:/Program Files/HomeBank/share/homebank/datas)

Feel free to send me a native file for your language to be shared in the next releases !
