Account window

The account window is used to manage one account at a time. It is the basis of every actions dealing with transactions.
Every action on transaction are done here, see the working with transaction for details.

Menu

The menu is the start of every actions on transactions. See here for a description.

Toolbar

The toolbar provides easy mouse shortcut for some of the menu. All toobutton have a tooltip that will help you know what action will be lanched when you click on the toolbutton.

Balance

The balance area contains the balance summary of the account, see this page for details. If the minor option is checked in the preferences, a minor toggle button appear here, see this page for details.

Transaction list

The list show you a summary of every transaction in the account. You can use the filter dialog if necessary. Or even change the sort order by clicking on the colum title.
Double-click works as follow, depending of the transaction selected:

  • one is selected: you open the transaction dialog in Edit mode
  • multiple are selected: while keep shift key pressed, you will edit and set the column last clicked and the dialog will be pre-filled with le column/transaction valueclicked on.

The status column indicate the change made to the transaction:
  • transaction added
  • transaction modified
  • transaction is validated
  • transaction is a remind

Date filter

The date filter will easily chnage the date range, without opening the filter dialog.

Date range

The date range display the actual range of date of the transaction filter.

Selection info

The statusbar at the bottom of the window display useful informations of the selected transactions:

  • no line or a single line is selected
    • show number of selected and hidden transactions (because of the filter)
  • 2 or more line are selected
    • - show number of selected and hidden transactions (because of the filter)
    - also balance (income - expense) of transaction selected
