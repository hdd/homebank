New features in v4.1

  • Automatic assignment of payee/category
  • New payment mode
  • Move for payee and category
  • Import process is more simple
  • Filter on plain text for info/description/tag field
  • Some new preference options
  • Some changes to the user interface

:: Automatic assignment of payee/category

This feature allows you to assign a payee and / or category from a word found in the Description field.

Note, however, HomeBank will affect if and only if the payee and the category are both empty.

There is a new dialog box called Assignments in which you create rules that will be used to automatically assign.
This automatic assignment comes at a time you import a file, or you can trigger it manually from the Accounts window.

:: New payment mode

HomeBank 4.1 introduce some more payment mode:

  • Debit card
  • Standing order
  • Electronic payment
  • Deposit
  • Financial institution fee

:: Move for payee and category

This feature allows to replace a payee/category by another. This will allow you to simply rearrange the payees and the categories.

:: Import process is more simple

HomeBank import Assistant now consist of only 3 steps instead of 6.

  1. The introduction page

    2. This page must be kept to follow the rules of the Gnome HIG

  2. The file selection page

    3. no changes on this page

  3. The import control page

    4. 3 steps were merged into this single page and you can now:

    • view the file information
    • control the account to import
    • control the transactions to import
    But you can also trust HomeBank and go directly to the confirmation page.

  4. The confirmation page

    5. This page is now more readable, and gives details on what will be chnaged in your account, following the import.

:: Filter on plain text for info/description/tag field

Now you can filter the transactions on a criterion of plain text for the following fields:

  • description
  • info
  • tag

This was a so long missing feature since the Linux port from the Amiga version.

:: Some new preference options

Here are the new preference options available in 4.1:

  • column order for transaction list
  • amount color preset (Tango)
  • show/hide the rule hint in every treeview
  • show/hide splash window when the program start
  • enable/disable colored amount

:: Some changes to the user interface

  • A new 'View' menu to enable/disable toolbar, statusbar and upcoming list
  • The size repartition between account and upcoming list in the main window is saved
  • The sort order in account window is saved
  • The recent menu is now in the toolbar
  • Menu were a little bit reworked
  • The selected item in tree view are more visible because of video inverted vs colored
