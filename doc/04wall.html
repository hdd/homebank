Wallet dialog

The wallet dialog is used to edit,modify and manage the wallet properties.

Dialog call

This dialog can be called in the following way:
  • from the mainwindow-menu: File / Properties

Using wallet dialog

General

Owner specify the title for the wallet, this will be used as the main window title.

Automatic transaction

Insert x days in the future

specify a number of days to add to the today's date (limit of insertion) when HomeBank insert automated transactions.
For example: you specify 30 days here, HomeBank will insert automated transactions that should occur till today + 30 days.

Car cost

Category

specify the default group for the CarCost.
