Automatic assignment

This function allow the assignment of payees and categories automatically to transactions.

  • Create assignment rules
  • When the auto assignment occurs ?
  • Notes

:: Create assignment rules

The managment can de done in the assignment dialog

You will create rules like:
if you find the word 'stuff' in the Description field of a transaction = assign payee A and / or category B to that transaction.

To create an assignment rule:

  • click the Add button
  • modify the search description text
  • optionaly check the case sensitive checkbox
  • select the payee and/or category to assign

:: When the auto assignment occurs ?

  • Everytime you import a file into HomeBank using the import assistant
  • You can trigger it manually from the Account window menu or toolbar

:: Notes

HomeBank will assign the payee and the category to a transaction if, and only if, both are empty.
