Automatic addition

This feature lets you program the addition of transaction automatically to their due date or in advance.

  • Create automatic transactions
  • When the automatic addition occurs ?
  • Control the add in advance
  • List of upcoming automatic transactions

:: Create automatic transactions

The management can be done in the archive dialog (more about archive here).

To create an automated transaction:

  • select an existing archive (or create a new one)
  • check the 'Activate' check-box in the 'Automated' frame
  • adjust the needed parameters:
    • frequency: every x day, week, month, year
    • next occurrence date
    • optional limitation to X occurrence

:: When automatic transactions occurs ?

  • Everytime you open your HomeBank file
  • You can trigger it manually from the Main window menu

:: Control the add in advance

By default, the transaction will be automatically added to accounts at their due date.

You can change this behavior and configure HomeBank for the transaction that be added a few days before their due date. This way you will know the future balance in advance for your accounts.

From the main window:

  • Select the file menu
  • Select the properties submenu
  • Adjust the 'Insert' xx 'days into the future

:: List of upcoming automatic transactions

The main window display a list of 'Upcoming automated transactions'.
This list displays every scheduled (automated) transaction and their state, especially:

  • next occurrence date
  • remaining days

This remaining day count is negative if the transaction insertion is overdue.
