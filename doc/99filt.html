Filter dialog

The filter dialog is used to adjust the filter for transaction from the account window and the statistics window.

Dialog call

This dialog can be called in the following way:

  • from the account window-menu: Tools / Filter...
  • from the account window-tool-bar.
  • from the statistics window-tool-bar.

Using filter dialog : General tab

Filter Date

Option

set this filter state: inactive, include, exclude

From

the date minimum value

To

the date maximum value

Filter State

Validated

set this filter state: inactive, include, exclude

Force display 'Added'

force transaction in 'added' state to always be displayed

Force display 'Edited'

force transaction in 'edited' state to always be displayed

Filter Amount

Option

set this filter state: inactive, include, exclude

From

the amount minimum value

To

the amount maximum value

Filter Payment

Option

set this filter state: inactive, include, exclude

payment

Using filter dialog : Other tabs (Category / Payee / Account)

These tabs all function-the same manner. You can filter these using a list of item, including or excluding the selected items.
The account tab is not showed when you filter from the account window, it is showed only from the statistics dialog.

Option

set this filter state: inactive, include, exclude

All

will select all the items in the list

None

will unselected all the items in the list
