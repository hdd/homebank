Filter dialog

The filter dialog is used to adjust the filter for transaction from the account window and the statistics window.

Dialog call

This dialog can be called in the following way:
  • from the account window-menu: Edit / Filter...
  • from the account window-toolbar.
  • from the statistics window-toolbar.

Using filter dialog : General tab

Filter Date

Option set this filter state: inactive, include, exclude
From the date minimum value
To the date maximum value

Filter State

Validated set this filter state: inactive, include, exclude
Force display 'Added' force transaction in 'added' state to always be displayed
Force display 'Edited' force transaction in 'edited' state to always be displayed

Filter Amount

Option set this filter state: inactive, include, exclude
From the amount minimum value
To the amount maximum value

Filter Payement

Option set this filter state: inactive, include, exclude
payement

Using filter dialog : Other tabs (Category / Payee / Account)

These tabs all functionthe same manner. You can filter these using a list of item, including or exluding the selected items.
The account tab is not showed when you filter from the account window, it is showed only from the statistics dialog.

Option set this filter state: inactive, include, exclude
All will select all the items in the list
None will unselect all the items in the list
