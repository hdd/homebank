Budget report

The budget report will summarize your running budget and will display its result.

See also: budget dialog, and basic concepts

Window call

This report can be called in the following way:
  • from the mainwindow-menu: Report / Budget
  • from the mainwindow-toolbar.

Using budget report

Display

Kind specify the item the result should be computed for: Expense, Income
View select the graph display: Spent, Budget or Decay

Date filter

From / To specify date bound limit to restrict the results to
Range / Month / Year fastly select a date with predefined range, or specific month and/or year

Toolbar

The toolbar is the main control of the display. All toobutton have a tooltip that will help you know what action will be launched when you click on the toolbutton.

Total

This is the total of columns of the listview for: Spent, Budget and Decay

Result list

The result list display the computed amounts according to the current selection.
