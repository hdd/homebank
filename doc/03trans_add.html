:: How to add transactions

:: By hand

From the main window or the account window

  • Select the Transaction menu
  • Select the Add submenu
  • Click on the Add toolbar button

Then you fill in all fields by hand. You can add categories and payee from the dialog.

:: From the bookmark

Same as above, but you'll fill those fields with a transaction from favorites listbox "Fill from transaction"

:: From inheritance

From the account window only:

  • select the transaction to inherit
  • click on the Inherit tool-bar button or use Transaction/Inherit menu
  • change the needed value
  • validate one or more iteration

:: How to edit transactions

:: Single

From the account window only:

  • double click the transaction to edit
  • select the transaction to edit
  • click on the Edit toolbar button or use Transaction/Edit menu

:: Multiple

From the account window only:

  • select several transactions (using the shift key)
  • Keep pressing the <shift> key, double click on a column
  • (the clicked transaction/column will fill the popup)
  • A pop-up will rise
  • Then you can change the value

:: Bookmark a transaction

Again, to avoid the manual entry, you can create an Archive from an existing transaction

From the account window only:

  • select one or more transactions
  • select the Transaction menu
  • select the Create an archive submenu

:: Remind a transaction

You can enter transactions to remind, for example, do not forget a debt to someone, or vice versa.
The transaction in remind state are completely ignored by all treatments.

To remind a transaction, just chek the remind toggle button in the transaction dialog.
