Add transactions

The transaction will be the heart of your accounting activity, so it is important to know exactly how HomeBank enable you to manipulate transaction and especially how can it make your life easier with that !

Here is a list of possible way to add transaction into HomeBank:

  • add manually, enter a specific transaction or use the bookmark (Archives)
  • inherit, which mean duplicate from an already existing transaction
  • import from a file from your bank or other software in QIF, OFX or CSV format,
  • schedule the recurrent transactions (Archives automated)

:: Add manually

This can be done from the main window and the account window

  • select the Transaction/Add menu
  • click on the Add tool-bar button.
    Note: you can then fill the transaction from an archive (bookmarked transaction)

:: Inherit (duplicate from a transaction)

This can be done from the account window only.

  • select the transaction to duplicate
  • click on the Inherit tool-bar button
  • change the needed value
  • validate one or more iteration

:: Tips & tricks

It is possible to create archives from existing transactions:
