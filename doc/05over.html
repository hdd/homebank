Overdrawn report

The overdrawn report is an analysis part focused on the balance evolution during the time. A specific purpose will be to visually control your overdrawn.

The transactions are displayed ordered by date in order for you to check the overdrawn according to the minimum set in the account window.

The results are displayed into a list or with a line chart, giving a visual view of the account flow.

Window call

This report can be called in the following way:
  • from the mainwindow-menu: Report / Overdrawn
  • from the mainwindow-toolbar.

Display

Account select the account to display result for.

Date filter

From / To specify date bound limit to restrict the results to
Range / Month / Year fastly select a date with predefined range, or specific month and/or year

Toolbar

The toolbar is the main control of the display. All toobutton have a tooltip that will help you know what action will be launched when you click on the toolbutton.

Infos

Inform you about the number of transaction in overdrawn, the total number of transaction and the value of the overdrawn forthis account.

Result list

The result list display the computed balances according to the current selection.
