Transactions dialog

The transaction dialog is used to add, edit and manage the transactions.
Read also the working with transactions to know more about HomeBank possibilities.

Dialog call

This dialog can be called in the following way:

  • from the mainwindow-menu: Transaction / Add...
  • from the mainwindow-toolbar.
  • from the account window-menu: Transaction / Add... / Inherit... / Edit...
  • from the accountwindow transaction list: by double-click on an transaction

Using transaction dialog

The 'Add' button enable to add an transaction and keep the transaction dialog open to add more transactions.

General infos

Date

date of the transaction (dd.mm.yy).
- use shift key + arrow up/down to increase/decrease date.
- the right side button popup a full calendar

Description

description of transaction.
and optionally vehicle cost data's, see vehicle cost.

Amount

amount of the transaction
- the right side +/- button toggle between income/expense

Account

account the transaction should be attached to.

Validated

when the transaction has been checked with your bank statement, you can mark transaction as 'validated'.
the amount will be then integrated in the bank balance.

Remind

when you are in add mode, if you set this option, an transaction of the opposite amount will be inserted also (marked in the list with an '!'), this is very useful when you lent money and don't want to forget about it. The remind transaction are never computed into the balances and are always displayed. an transaction can be a transparent reminder when this is checked. See working with transaction.

Optional infos

Payment

payment for the transaction. It will be displayed as small icons. Possible values are:

  • (none)
  • credit card
  • check
  • cash
  • transfer
  • internal transfer
  • debit card
  • standing order
  • electronic payment
  • deposit
  • fi fee
