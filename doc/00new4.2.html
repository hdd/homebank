New features in v4.2

  • Trend time report
  • Chart X-zoom
  • OFX import of memo

:: Trend time report

This new report permit to view the trend in time for account, category and payee.
You can display the result by day, week, month, quarter and year.
Adding to this there is an option to cumulate the result, and to display the result for the complete wallet.

:: Chart X-zoom

You can now adjust dynamically the interval between the displayed items for line and bars charts in every reports.
This will permit to condensate the chart when it is very large, or enlarge it when necessary.

:: OFX import of memo

The memo field for OFX/QFX files, can now be imported according to the preferences defined by the user.
There is a new item "Import" in the preference that defines what to do with this field:

  • ignore
  • add to the description
  • add to the info

Thus, at every import, the memo field will correspond exactly to your needs.
